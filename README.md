
# Descripción

Este proyecto muestra cómo realizar operaciones CRUD sobre una base de datos utilizando Entity Framework como capa de acceso a datos.

Las operaciones implementadas incluyen:

- Crear registros en la base de datos
- Leer datos almacenados
- Actualizar registros existentes
- Eliminar registros

Entity Framework actúa como un ORM, permitiendo trabajar con la base de datos mediante objetos de C# en lugar de escribir consultas SQL manuales.



# Requisitos

Antes de ejecutar el proyecto debes tener instalado:

- Visual Studio
- .NET Framework / .NET SDK
- SQL Server
- SQL Server Management Studio (opcional)
